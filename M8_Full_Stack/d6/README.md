### Getting Started


1. Install pipenv 
```shell
 
 pip3 install pipenv

```
2. Install dependencies to env
```shell
 
 pipenv install SQLAlchemy python-dotenv psycopg2-binary , flask

```
3. Activate env


```shell
 
 pipenv shell

```


4. Run 


```shell
 
 python main.py

```
